## Usage

With Yarn:

```
yarn create react-boilerplates
```

With PNPM:

```
pnpm create react-boilerplates
```

With NPM:

```
npm create react-boilerplates@latest
```

Then follow the prompts and run the project.  

For example, using yarn.

```
cd <your-project-name>

yarn

yarn dev
```
